◆ InicjalizujTest()

bool BazaTestu::InicjalizujTest ( const char *  sNazwaTestu)

Inicjalizuje test kojarzac zmienna dostepna poprzez wskaznik wskBazaTestu z dana tablica wyrazen, ktora reprezentuje jeden z dwoch dopuszczalnych testow.

Parametry
sNazwaTestu- wskaznik na napis wybranego typu testu.
 @pre Parametr wskBazaTestu nie moze byc pustym wskaznikiem. Musi zawierac adres
   zmiennej reprezentujacej baze testu, ktora wczesniej zostal poprawnie
   zainicjalizowany poprzez wywolanie funkcji InicjalizujTest.
 @pre Parametr sNazwaTestu musi wskazywac na jedna z nazw tzn. "latwe" lub "trudne".
Zwraca
bool
Zwracane wartości
true- gdy operacja sie powiedzie i test zostanie poprawnie zainicjalizowany,
false- w przypadku przeciwnym.

unsigned int  IloscPytan

W bazie testu ustawia wybrany test jako biezacy test i indeks pytania ustawia na pierwsze z nich.

Parametry
wskTabTestu- wskaznik na tablice zawierajaca wyrazenia arytmetyczne bedace przedmiotem testu,
IloscPytan- ilosc pytan w tablicy.
Warunek wstępny
Parametr wskBazaTestu nie moze byc pustym wskaznikiem. Musi zawierac adres zmiennej reprezentujacej baze testu, ktora wczesniej zostal poprawnie zainicjalizowany poprzez wywolanie funkcji InicjalizujTest.
Parametr wskTabTestu zawiera wskaznik na istniejaca tablice.
Parametr IloscPytan zawiera wartosc, ktora nie przekracza ilosci elementow w tablicy dostepnej poprzez wskTabTestu.
